## Deploy Decision Record

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Reverse Proxy | Nginx native **or** NPM | NPM recommended cho @itadmin quen GUI; native cho lightweight |
| SSL | Let's Encrypt (Certbot / NPM) | Free, auto-renew, không cần mua cert |
| Process Manager | systemd | Ubuntu standard, auto-restart, journalctl logging |
| GPU Startup Gate | `runtime_device == "cuda"` | Mandate từ CPO — không cho traffic nếu GPU unavailable |
| Frontend Port | `3005` | Tránh conflict với Open WebUI (3000) trên S1 per IT Admin allocation |
| API Port | `8000` | `HOST=127.0.0.1` enforced by @itadmin — localhost-only |
| Public ingress | `80/443` | Nginx/NPM reverse proxy, no direct app port exposure |
| Pre-deploy check | Port 3005 conflict | Hard-fail nếu 3005 đã bị chiếm (CPO `2.txt:984-1015`) |
| Post-deploy smoke | 4-check checklist | CPO approved: frontend + backend GPU + HTTP(301) + HTTPS(mflux_reachable) |
| Port freeze | `3005` final | No further port changes without CPO approval (Round 2 `2.txt:984-1015`)
| Live bridge | `172.19.0.0/16 -> 3005/tcp` | UFW allow required because OGA app runs natively while NPM runs in Docker |
