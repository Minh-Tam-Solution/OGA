# User Stories — NQH Creative Studio Sprint 1

## Overview

These user stories define the **G1-level acceptance criteria** for Sprint 1 of the NQH Creative
Studio fork. Each story maps to one or more Functional Requirements (FR-*) in
`docs/01-planning/requirements.md` and is bounded by `docs/01-planning/scope.md`.

**Target personas:**
- **Employee** — marketing, design, or content team member at NQH/MTS; uses the studio daily to
  create images for internal and external campaigns.
- **IT Admin (dvhiep)** — provisions and maintains the Mac mini M4 Pro 24 GB inference server;
  deploys and configures the application on the LAN.
